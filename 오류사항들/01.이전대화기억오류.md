# 오류 보고서: AI 에이전트 이전 대화(Context) 기억 상실 문제

## 🚨 문제 상황 (Problem)
Streamlit 웹 UI의 챗봇과 대화 시, 사용자가 이전 대화의 맥락을 생략하고 대명사나 후속 지시를 내리면 AI가 이전 상황을 전혀 기억하지 못하고 되묻는 현상 발생.

**[재현 시나리오]**
- **User:** "1구역 가스터빈 (T-100)의 상태를 점검완료로 바꿔줘"
- **AI:** "죄송하지만 '점검완료' 상태로는 변경할 수 없습니다. 가이드라인에 따라 '점검 요망' 또는 '수리 중'으로만 상태를 변경할 수 있습니다."
- **User:** "그럼 수리중으로 바꿔" (어떤 장비인지 주어가 빠진 맥락 의존형 질문)
- **AI:** "어떤 설비의 상태를 변경할까요?" (이전에 T-100에 대해 이야기하던 것을 까먹음)

## 🔍 원인 분석 (Cause)
Streamlit은 입력된 명령을 처리할 때 마다 전체 페이지를 다시 그리며 동작함(Stateless). 따라서 코드를 작성할 때, 사용자가 방금 친 1개의 문장(`prompt`)만 LangChain의 LLM 모델에 단독으로 전송하게 구현되어 있었음. GPT나 Gemini와 같은 언어 모델은 API로 단일 문장을 받을 경우 이전 대화 기록을 자동으로 알 수 없으므로, 매번 처음 만난 사람처럼 답변하게 됨.

## 🛠️ 해결 조치 (Solution)
Streamlit 자체의 캐시 시스템인 `st.session_state`에 저장되어 있는 **이전의 모든 대화 기록(Context)**을 가져와서, LangChain이 이해할 수 있는 객체인 `HumanMessage(사람 질문)`와 `AIMessage(AI 답변)` 리스트 형태로 엮음. 
그 후, 달랑 한 문장이 아니라 **"전체 대화 히스토리" 뭉치**를 한 번에 Agent(LLM)에게 전달하도록 코드를 수정하여 완벽한 기억력을 부여함.

## 💻 코드 수정 내역 (Code Changes)

**수정 파일:** `C:\YoulSystem\app.py`

### [Before] 수정 전 코드 (단일 문장만 전달)
```python
from langchain_core.messages import HumanMessage

# (중략) ...

        # AI 생각 및 행동 시작
        with chat_container.chat_message("assistant"):
            with st.spinner("AI가 매뉴얼 검색 및 시스템 제어 중입니다... ⚙️"):
                try:
                    # ❌ 여기서 문제 발생: 방금 사용자가 친 1문장(prompt)만 덩그러니 보냄
                    response = agent_executor.invoke({"messages": [HumanMessage(content=prompt)]})
                    answer = response["messages"][-1].content
                except Exception as e:
                    answer = f"시스템 오류: {e}"
```

### [After] 수정 후 코드 (전체 대화 히스토리 전달)
```python
# 1. AIMessage 모듈을 사용하기 위해 패키지 Import 구문에 추가
from langchain_core.messages import HumanMessage, AIMessage

# (중략) ...

        # AI 생각 및 행동 시작
        with chat_container.chat_message("assistant"):
            with st.spinner("AI가 매뉴얼 검색 및 시스템 제어 중입니다... ⚙️"):
                try:
                    # ✅ [해결 로직 추가] 
                    # Streamlit 화면 상에서 여태 나눴던 모든 대화 내용을 
                    # LangChain이 이해하는 사람(Human)/봇(AI) 메시지 덩어리로 변환
                    langchain_messages = []
                    for m in st.session_state.messages:
                        if m["role"] == "user":
                            langchain_messages.append(HumanMessage(content=m["content"]))
                        elif m["role"] == "assistant":
                            langchain_messages.append(AIMessage(content=m["content"]))
                            
                    # ❌ Agent 호출 시 단일 문장이 아닌, [전체 누적 대화 리스트]를 전달함
                    response = agent_executor.invoke({"messages": langchain_messages})
                    answer = response["messages"][-1].content
                except Exception as e:
                    answer = f"시스템 오류: {e}"
```
